Вариант № 14: «Стопка Работ»
Простой класс: Работа студента – определяется фамилией студента (строка), оценкой (от 2
до 5, также оценка может быть ещё не выставлена), номером первого листа и номером
последнего листа. Полностью сшитая работа начинается первым листом и заканчивается
последним, однако в некоторых случаях разные листы работы могут оказаться не
скреплены. В этом случае, вместо одной работы в стопке окажется несколько работ для
одного студента (одна начинается с листа 1 и заканчивается листом n, вторая начинается с
листа n+1 и заканчивается листом m и т.д.). При этом не гарантируется, что разные части
одной работы будут лежать в стопке рядом или в правильной последовательности. Оценка
может присутствовать только на 1-м листе (то есть на работе, начинающейся с 1-го листа).
Методы простого класса (помимо общих): • создание экземпляра класса с инициализацией
фамилией студента, оценкой, номером первого и номером последнего листов; • создание
экземпляра класса с инициализацией только фамилией студента (работа из одного листа без
оценки); • (+) объединение двух частей работы в одну, при этом объединить можно только
подряд идущие части работы одного студента (требуется соблюдать свойство
коммутативности); • разбиение работы на листы, результатом является массив работ (по
одному листу каждая); • (<=>) сравнение двух работ сперва по фамилии, затем по номеру
первого листа; • выставление оценки (с необходимыми проверками).
Сложный класс: Стопка работ – определяется множеством работ студентов, хранимым при
помощи структуры данных стек.
Методы сложного класса (помимо общих): • создание экземпляров класса с инициализацией
заданным количеством работ из массива работ; • (+=) добавление новой работы на вершину
стопки; • извлечение одной работы с вершины стопки; • проверка состояния стека (пуст,
частично заполнен или полон); • объединение всех разбитых на части работ (то есть после
выполнения операции для каждого студента должно остаться только по одной работе, если
это возможно); • разбиение всех работ в стопке на отдельные листы; • извлечение
следующей работы без оценки.
